# AnonAsk — 向我提问

> 一个轻量的匿名问答工具。注册后获得个人链接，发到朋友圈征集问题，在收件箱里一一回答。前端完全匿名，后台记录完整。

**项目状态**：个人自用项目，开源展示。

---

## 这是什么？

AnonAsk 的工作方式：

1. **注册/登录** — 用手机号 / QQ号 / 微信号 + 密码注册
2. **分享链接** — 获得个人页面 `/u.php?uid=xxxxxxxxxxxx` 分享到朋友圈
3. **别人向你提问** — 点开链接 → 登录 → 匿名向你提问
4. **你回答案** → 在收件箱里看到问题，一一作答

**前端完全匿名**：所有问题和回答的前端展示均不显示任何用户信息（提问者、回答者）。但后台数据库记录完整，开发者可查。

---

## 核心逻辑

| 角色 | 做什么 | 前端可见性 |
|------|--------|-----------|
| 链接主人（你） | 分享链接，在收件箱回答问题 | 可见问题和回答内容 |
| 提问者 | 登录后向链接主人提问 | 仅可见自己的问题和回答 |
| 浏览者 | 打开链接查看问答 | 只读，可见问题和回答内容 |

**关键设计**：
- 所有用户必须登录才能提问或回答
- 前端永不展示任何用户标识（头像、昵称、UID）
- 后台完整记录 `author_uid` / `target_uid`，开发者可查

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | HTML + CSS + JavaScript（纯静态）|

| Web 服务器 | Nginx |

## 项目结构

```
AnonAsk/
├── index.php                 # 首页入口
├── u.php                     # 用户公开页路由
├── sql/schema.sql            # 数据库建表脚本
│
├── includes/
│   ├── config.php            # 配置（数据库/限流/密码规则）
│   ├── db.php                # PDO 数据库连接
│   └── functions.php         # 会话管理 + 工具函数
│
├── api/
│   ├── auth.php              # 注册 / 登录 / 注销 / 检查状态
│   ├── question.php          # 提问（向某人）+ 查询问答列表
│   ├── answer.php            # 回答问题 / 删除问答
│   └── rate-limit.php        # IP + UID 双维度频率限制
│
├── pages/
│   ├── login.php             # 登录 / 注册页
│   ├── dashboard.php         # 用户收件箱（查看问题 / 回答 / 删除）
│   └── u.php                 # 用户公开页（显示问答 + 提问入口）
│
└── assets/
    ├── css/style.css         # 公共样式
    └── js/app.js             # 公共 JS（认证 / API / UI 工具）
```

## 数据库表

| 表 | 说明 |
|----|------|
| `users` | 用户（UID / 联系方式 / 密码）|
| `questions` | 问题（`target_uid` 发给谁，`author_uid` 谁问的）|
| `answers` | 回答（`question_id` 对应问题，`author_uid` 回答者）|
| `rate_limits` | 频率限制日志 |

---

## 功能范围

- ✅ 注册 / 登录（手机号 / QQ号 / 微信号 + 密码）
- ✅ 12 位纯数字 UID
- ✅ 个人链接页 `/u.php?uid={uid}`
- ✅ 向对方匿名提问（需登录）
- ✅ 收件箱回答问题
- ✅ 删除问答
- ✅ 前端完全匿名（不显示任何用户信息）
- ✅ IP + UID 双维度防刷

---

## 本地运行

1. 导入 `sql/schema.sql` 到 MySQL
2. 修改 `includes/config.php` 中的数据库连接信息
3. 配置 Nginx 指向项目根目录
4. 访问首页完成注册即可使用
