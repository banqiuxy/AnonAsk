# AnonAsk - 完全匿名的你问我答

> 一个极简的匿名问答工具 —— 不需要登录，不需要手机号，不需要邮箱。每个人都是匿名的，每个问题都有自己的独立页面。

**项目状态**：个人 Demo 项目，开源展示，不提供编译或部署指南。

---

## 🎯 这个项目是什么？

AnonAsk 是一个**给自己用的**匿名问答小工具。它的工作方式很简单：

1. 你访问网站 → 自动获得一个匿名身份 ID（存在浏览器里）
2. 你可以提出一个问题 → 系统生成一个唯一的问题 ID
3. 把问题 ID 对应的链接发给任何人 → 对方打开就能看到这个问题和所有回答
4. 任何人（包括你自己）都可以在问题下匿名回答

**这不是一个社区平台**。没有广场页，没有用户关注，没有消息推送。每个问题都是独立的“小纸条”。

---

## ✨ 核心逻辑

| 角色 | 做什么 | 怎么识别 |
|------|--------|----------|
| 提问者 | 提出问题 | 浏览器生成的匿名 ID（持久化存储） |
| 回答者 | 回答问题 | 独立的匿名 ID（与提问者 ID 不同） |
| 浏览者 | 通过链接查看问答 | 无需任何 ID，只读 |

**关键设计**：
- 每个问题拥有唯一访问链接 `/question/{id}`
- 提问者和回答者的匿名身份**完全独立**，互不关联
- 没有用户注册，没有登录，没有任何个人信息收集

## 🧱 技术栈

| 层 | 技术 |
|----|------|
| 前端 | Node.js（用于开发服务器/代理）|
| 后端 | PHP |
| 数据库 | MySQL |

> 为什么是纯 PHP？因为不需要框架，不需要路由库，原生 PHP 足够处理这几个简单的接口。
